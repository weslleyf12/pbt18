Este aplicativo tem como finalidade o envio de imagens a um servidor, para posteriores classificações. No campo para se especificar uma URL, utiliza-se a URL do servidor que leve até algum arquivo .php ao qual encaminhará a imagem para um diretório específico. Ex: "54.1.0.105:80/teste/index.php".

O campo onde especifica-se a frequência serve para estabelecer de quanto em quantos segundos a foto será tirada e enviada. Não recomenda-se utilizar uma frequência pequena, pois pode compro meter o desempenho do aplicativo.
